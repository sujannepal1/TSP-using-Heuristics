# Traveling Salesman Problem (TSP) using Nearest Neighbor + 2-Opt Optimization

This project implements a solution to the **Traveling Salesman Problem (TSP)** using:

1. **Nearest Neighbor Algorithm** — to generate an initial feasible route
2. **2-Opt Algorithm** — to optimize and improve the route distance

The program also visualizes both the initial and optimized routes using **NetworkX** and **Matplotlib**.

---

## 📌 Problem Description

Given a set of cities and distances between them, the goal is to:

* Start from a random city
* Visit each city exactly once
* Return to the starting city
* Minimize the total travel distance

This is a classic NP-Hard optimization problem.
